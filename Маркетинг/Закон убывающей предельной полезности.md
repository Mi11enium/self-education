# Закон уменьшения предельной полезности (DMU)

По мере употребления количества данного товара, полезность которую мы от него получаем продолжает уменьшаться

Units
TU (Total utility)
MU (Marginal utility)

---
При прорисовке графика становиться видно, что:
1. TU = max -> MU = 0
2. TU = neg -> MU = neg
